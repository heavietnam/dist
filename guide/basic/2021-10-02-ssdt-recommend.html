SSDT-Recommned

Những SSDT khuyên dùng cho các phần cứng

Võ Nguyễn Hoàng Long
2021-10-02
10 min

Đối với hầu hết các model chỉ cần SSDT-EC là đã có thể boot lên. Tuy nhiên để sử được đầy đủ chức năng thì các bạn sẽ cần add đầy đủ các SSDT. Sau đây là 1 sô ssdt khuyến khích nên có:

# Desktop

# Penryn/Lynnfield/Clarkdale

  • SSDT-EC

# SandyBridge/Ivy Bridge

  • CPU-PM
  • SSDT-EC

# Haswell/Broadwell

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC

# Skylake/Kaby Lake

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC

# Coffee Lake

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-AWAC ( dùng ssdt-time để prebuilt ssdt )
  • SSDT-PMC

# Comet Lake

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-AWAC ( dùng ssdt-time để prebuilt ssdt )
  • SSDT-RHUB

# AMD (17/19h)

  • SSDT-CPUR

# High End Desktop

# Nehalem and Westmere

  • SSDT-EC

# Sandy Bridge-E/Ivy Bridge-E

  • SSDT-EC
  • SSDT-UNC

# Haswell-E/Broadwell-E

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-RTC0-RANGE
  • SSDT-UNC

# Skylake-X

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-RTC0-RANGE

# Laptop

# Clarksfield and Arrandale

  • SSDT-EC
  • SSDT-PNLF
  • SSDT-IRQ

# SandyBridge/Ivy Bridge

  • SSDT-PM
  • SSDT-EC
  • SSDT-PNLF
  • SSDT-IRQ
  • SSDT-IMEI

# Haswell/Broadwell

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-PNLF
  • SSDT-IRQ
  • SSDT-XOSI ( hoặc SSDT-GPI0 )

# Skylake/Kaby Lake

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-PNLF
  • SSDT-XOSI ( hoặc SSDT-GPI0 )

# Coffee Lake/Whiskey Lake/Comet Lake

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-PNLF
  • SSDT-XOSI ( hoặc SSDT-GPI0 )
  • SSDT-AWAC ( dùng ssdt-time để prebuilt ssdt )
  • SSDT-PMC ( chỉ sử dụng cho coffelake gen 9 )

# Ice Lake

  • SSDT-PLUG ( hoặc SSDT-PM )
  • SSDT-EC
  • SSDT-PNLF
  • SSDT-XOSI ( hoặc SSDT-GPI0 )
  • SSDT-AWAC ( dùng ssdt-time để prebuilt ssdt )
  • SSDT-RHUB

Lưu ý: đối với ssdt-awac các bạn có thay thế bằng cách sử dụng fix rtc
